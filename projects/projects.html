Asteroids

Java

A clone of the classic Asteroids arcade game built in Java using the Swing graphics library.

  • Implemented a complete Asteroids clone from scratch in Java, including input handling, rendering, collision detection, and game-state management.
  • Used object-oriented design patterns (inheritance, polymorphism, encapsulation, abstract base classes) to model game entities and behaviors cleanly.
  • Designed a game loop using Java concurrency primitives, leveraging Runnable and timers for consistent updates and input handling.
